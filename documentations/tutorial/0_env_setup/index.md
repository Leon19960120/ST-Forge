# 环境配置

本章节带你从零搭建 STM32 开发环境。教程内容按**开发平台**分别提供，请根据你的系统选择对应的路径：

- 🪟 **Windows** — 原生 Windows 工具链与驱动安装，适合不使用 WSL 的同学：[Windows 路径](./win/01_environment_setup)
- 🐧 **WSL (Windows Subsystem for Linux)** — 在 WSL2 中使用 Linux 工具链开发，体验更接近生产环境：[WSL 路径](./wsl/01_environment_setup)

> 两条路径的章节结构完全一致（环境配置 → 项目结构 → 代码详解 → 编译烧录 → 调试 → 基础概念 → 硬件附录），按顺序学习即可。如果你不确定选哪个，推荐先从 **Windows** 路径开始。

完整章节索引见 [教程首页](../)。
