# 环境配置

本章节带你从零搭建 STM32 开发环境。教程内容按**开发平台**分别提供，请根据你的系统选择对应的路径：

- 🪟 **Windows** — 原生 Windows 工具链与驱动安装，适合不使用 WSL 的同学：[进入 Windows 路径 →](./win/)
- 🐧 **WSL (Windows Subsystem for Linux)** — 在 WSL2 中使用 Linux 工具链开发，体验更接近生产环境：[进入 WSL 路径 →](./wsl/)

> **为什么侧边栏里 Windows 与 WSL 两路径的章节标题看起来一样？**
> 这是按**开发平台**有意拆分的两条路径，章节结构一致（环境配置 → 项目结构 → 代码详解 → 编译烧录 → 调试 → 基础概念 → 硬件附录），但每篇内容都针对各自平台（工具链、驱动、USB 权限、命令都不同）。**按你的系统选一条路径走完即可，不需要两套都看。** 如果不确定选哪个，推荐先从 **Windows** 路径开始。

完整章节索引见 [教程首页](../)。
